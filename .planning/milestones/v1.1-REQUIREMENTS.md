# Requirements: AURA — Marco v1.1 (Polimento UAT + Multi-tenancy + Automação)

**Defined:** 2026-05-11
**Shipped:** 2026-05-15
**Core Value:** Um colaborador monta orçamento do zero ao PDF com dados organizados por arquiteto; admin controla preços, pedidos e filtragem sem planilha paralela.

> **Contexto:** 18 requirements em 6 categorias derivadas do UAT pessoal do Lenny em prod (2026-05-11) após arquivamento do v1.0. Marco focado em opcionalizar friction points de cadastro, isolar dados por colaborador (RLS), permitir edição de wizard, lapidar PDF/dashboard e abrir primeira automação assíncrona (aniversário).

### AUTO — Automação aniversário

- [x] **AUTO-01:** Sistema envia email 5 dias antes do aniversário do cliente para o colaborador dono → **DELIVERED** (Phase 12 — 2026-05-15)
- [~] **AUTO-02 (DELIVERED with deviation):** Sistema envia email 5 dias antes do aniversário do cliente para admin → **DELIVERED with deviation** (Phase 12 — 2026-05-15)
  - **Spec original:** email para o admin "David Grabarz" (email fixo, configurável via env)
  - **Entregue:** email para **todos os admins** via `has_role(admin)` dinâmico — RPC `buscar_admins_emails()` SECURITY DEFINER faz JOIN `auth.users + user_roles` e retorna emails dos admins ativos
  - **Justificativa (STATE.md Phase 12 D-22):** suporta N admins sem redeploy, evita hardcode de email no código/env, e cobre 100% do intent original ("admin recebe email"). Mesmo padrão usado pra substituir o `ADMIN_EMAIL` legacy do `request-access`. Sem prejuízo funcional — Lenny + Lucas confirmados no smoke 12-02.
  - **Dedup follow-up:** `toList` na edge fn pode duplicar quando owner=admin (ex: Lenny criando cliente e sendo admin). Mitigação trivial deferida pra v1.2+ (1 linha: `Array.from(new Set([colab_email, ...admin_emails]))`).
- [x] **AUTO-03:** Schema aditivo: campo `data_nascimento DATE` em `clientes` + cron pg_cron + edge function chamando Resend + tabela `aniversario_envios` com UNIQUE(cliente_id, ano_referencia) → **DELIVERED** (Phase 7 — schema; Phase 12 — automation)
